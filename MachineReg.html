ระบบทะเบียนเครื่องจักร
ชื่อผู้ใช้หรือรหัสผ่านไม่ถูกต้อง
วิศวกร: เข้าถึงข้อมูลทั้งหมดและฟีเจอร์ครบ
ช่าง: เข้าถึงข้อมูลซ่อมและประวัติเครื่องพื้นฐาน

ตั้งค่า Firebase (ซิงค์ข้อมูล)

กรอก Firebase Config เพื่อซิงค์ข้อมูลแบบ Real-time ระหว่างทุกเครื่อง ข้อมูลจะอัปเดตอัตโนมัติเมื่อใช้งานจากหลายเครื่องพร้อมกัน

วิธีสร้าง Firebase:
1. ไปที่ console.firebase.google.com → สร้าง Project ใหม่
2. เปิด Realtime Database → Create Database → ตั้ง Rules เป็น read/write: true
3. ไปที่ Project Settings → เลือก Web app → Copy config มาวาง
หากไม่ต้องการซิงค์ข้ามเครื่อง ข้ามการตั้งค่านี้ได้ ข้อมูลจะบันทึกใน browser นี้เครื่องเดียว

ตั้งค่าแจ้งเตือน Gmail

ระบบจะส่งอีเมลแจ้งเตือนเมื่อมีแผน PM ใกล้ถึงกำหนดหรือเกินกำหนด โดยใช้ EmailJS (ฟรี 200 อีเมล/เดือน)

วิธีตั้งค่า EmailJS:
1. สมัคร emailjs.com → Add Email Service (Gmail)
2. Create Email Template — ใส่ตัวแปร {"{{to_email}}"} {"{{subject}}"} {"{{message}}"}
3. คัดลอก Service ID, Template ID, Public Key มาวาง
